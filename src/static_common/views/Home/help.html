1. Login Page

Figure 1. The Login page

1.1 Login

To login, first input an email and password, then select the Login   button.

1.2 Remember me

Selecting the Remember Me? button stores the inputted email and password combination for future use, allowing the user to login without having to input email or password beginning the next time that the site is accessed.

2. Navigation Bar

Figure 2. The navigation bar

The Navigation Bar refers to the list of options found on the vertical, black bar on the left-hand side of the page. It allows for quick and simple navigation among the six primary sections of Upgrid:

  •   Dashboard
  •   "Whoops!" Report
  •   Enhancement Report
  •   Report Manager
  •   User Profile
  •   Client Support

3. “ Whoops!” Report

Figure 3. The "Whoops!" Report page

The “Whoops!” Report page allows for the viewing and manipulation of ‘Whoops Report’ program reports, which displays flaws detected on the user’s website content. The program reports can be manipulated in the following manner: individually, by selecting view or share on the “Whoops!” Report page. (see Section 4.2 Selecting and manipulating program reports) or, in batches, by adding reports to the Report Manager for the printing or sharing of report compilations. (see Section 5.1 Creating report compilations).

3.1. Progress circle

Figure 4. The progress circle

The progress circle displays the number of reports currently available or being processed.The table contains four programs in total, and only two of these four programs’ reviews has been completed.

3.2. Program list

Figure 5. The program list

Displays the list of programs

3.3. Review status

Figure 6. The review status

The drop-down STATUS menu allows the user to check the status of completion of their reports—selecting Complete displays those that are completed and selecting In Progress displays those that are still under review. If the status shows , it means the program is still under review. If the status shows , it means the program review is completed and the website of the program is error-free.

3.4. Search bar

Figure 7. The search bar

The Search allows for the searching of the client’s database of programs, so that programs can be located with ease (and/or sorted by degree).

4. Enhancement Report

Figure 8. The Enhancement Report page

The Enhancement Report allows for the viewing and manipulation of program reports. The program reports can be manipulated in the following manner: individually, by selecting view, downloading, or share on the Enhancement Report page. (see Section 4.2 Selecting and manipulating program reports) or, in batches, by adding reports to the ‘Reports Manager’ for the printing or sharing of report compilations. (see Section 5.1 Creating report compilations)

4.1. Confirming desired programs

First-time users will be prompted to confirm their desired programs. To do so, select the appropriate checkboxes to the left of each program to confirm the desired programs. (Programs must first be confirmed before moving on to the Enhancement Report.)

Figure 9. The enhancement program confirmation page

4.2. Selecting and manipulating program reports

a. The programs in the Program column represent those of the user’s own institution. Competing programs are indexed according to the priority level determined previously during contract negotiation between Gridology and the user’s institution.

Figure 10. The Enhancement Report table

b. To the right of the competing programs column there are two buttons:  View and  Share.

Figure 11. The Enhancement Report status

Selecting the  View button displays the Enhancement Report. Selecting the  Share button displays the URL link for the program’s Enhancement Report.

Figure 12. The Enhancement Report

5. Report Manager

The Report Manager stores and creates a compilation of the files selected in the "Whoops!” Report and Enhancement Report sections and allows for the printing or sharing of said compilation.

5.1. Creating report compilations

Select the check box to the left of each program will cause an icon to appear in the navigation bar to the right of the Report Manager indicating the number of files currently stored in the Report Manager.

Figure 13. Here the 'Applied Linguistics' report has been selected to be added to the ‘Report Manager'

5.2. Reports manager review

Figure 14. The 'Report Manager' page

a. Select the View button to view all the reports together as a single report.

b. To share the reports you have selected, please click the Share button.
