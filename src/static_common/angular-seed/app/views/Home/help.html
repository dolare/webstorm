1. Login Page

Figure 1, The Login page

1.1 Login

To login, first input an email and password, then select the “login” button.

1.2 Remember Me

Selecting the “Remember Me?” button stores the inputted email and password combination for future use, allowing the user to log in without having to input email or password beginning the next time that the site is accessed


2. Navigation Bar (Item 1 in Fig. 2, below)

The Navigation Bar refers to the list of options found on the vertical, black bar on the left-hand side of the page. It allows for quick and simple navigation among the five primary sections of Upgrid: the ‘“Whoops!” Report,’ ‘Enhancement Report,’ ‘Reports Manager,’ ‘User Profile,’ and ‘Client Support.’

Figure 2, The “Whoops!” Report

3. “ Whoops!” Report

The ‘“Whoops!” Report’ page allows for the viewing and manipulation of ‘Whoops Report’ program reports, which displays flaws detected on the user’s website content. The program reports can be manipulated in the following manner: individually, by selecting view, download, or share on the ‘“Whoops!” Report page. (see Section 4.2 Selecting and Manipulating Program Reports) or, in batches, by adding reports to the ‘Reports Manager’ for the printing or sharing of report compilations. (see Section 5.1 Creating Report Compilations).

3.1. Progress Bar (Item 2 in Fig. 2, above)

The Progress Bar displays the number of reports currently available or being processed.The table contains two programs in total, and only one of these two programs’ reviews has been completed.

3.2. Program list (Item 3 in Fig 2, above)

Displays the list of programs

3.3. Review Status (Item 4 in Fig 2, above)

The drop-down ‘STATUS’ menu allows the user to check the status of completion of their reports—selecting ‘completed’ displays those that are completed and selecting ‘in progress’ displays those that are still under review. If the status shows “In progress” it means the program is still under review. If the status shows , it means the program review is completed and the website of the program is error-free.

3.4. Search Bar (Item 5 in Fig 2, above)

The Search allows for the searching of the client’s database of programs, so that programs can be located with ease (and/or sorted by degree).


4. Enhancement Report

The ‘Enhancement Report’ allows for the viewing and manipulation of program reports. The program reports can be manipulated in the following manner: individually, by selecting view,, downloading, or share on the ‘Enhancement Report’ page. (see Section 4.2 Selecting and Manipulating Program Reports) or, in batches, by adding reports to the ‘Reports Manager’ for the printing or sharing of report compilations. (see Section 5.1 Creating Report Compilations)

First-time users will be prompted to confirm their desired programs. To do so, select the appropriate checkboxes to the left of each program to confirm the desired programs. (Programs must first be confirmed moving on to the enhancement report.)

5. Report Manager

The ‘Report Manager’ stores and creates a compilation of the files selected in the ‘“Whoops!” Report’ and ‘Enhancement Report’ sections and allows for the printing or sharing of said compilation.

Select the check box to the left of each program will cause an icon to appear in the navigation bar to the right of the ‘Report Manager’ indicating the number of files currently stored in the ‘Report Manager.’
